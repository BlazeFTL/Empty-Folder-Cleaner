# Empty Folder Cleaner

A lightweight Android utility that scans and removes empty folders cluttering your internal storage and SD card.

## Features

- **Delete Empty Folders - INTERNAL** — clean empty directories on internal storage
- **Delete Empty Folders - SD Card** — clean empty directories on external SD card
- **Scan Mode** — find empty folders without deleting them
- **Live Log** — real-time deletion log with per-folder status (`STATUS` / `DELETED`)
- **Deletion Summary** — scanned folder count, folders deleted, and execution time

## Folder Settings

Configurable engine parameters for the search algorithm:

| Setting | Description |
|---|---|
| Clean Android Folders | Scan/clean folders inside the Android system folder |
| Root Access | Grant SU root access for system directories |
| External Storage Location | Scan a custom external storage location |

> Enabling deep system paths can slow down execution but exposes isolated junk caches.

## Customization

6 accent color themes: Breeze Blue, Emerald Forest, Warm Amber, Royal Purple, Blaze Crimson, Ocean Teal.
